# TareaMEmentoComando
## Una empresa de software educativo desea una calculadora de escritorio que, además de las operaciones aritméticas básicas, permita deshacer y rehacer cualquier secuencia de cálculos sin perder la trazabilidad de los pasos realizados por el usuario. La calculadora debe soportar suma, resta, multiplicación, división y borrado del resultado actual. El sistema debe mantener un historial de comandos para deshacer y rehacer sin límite explícito de pasos (hasta que se cierre la sesión). Cada punto del historial debe almacenar el valor numérico y cualquier otra información necesaria para restaurar la calculadora exactamente al estado previo.
